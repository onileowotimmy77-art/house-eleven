# House Eleven AI Context

Every AI assistant working on House Eleven must follow these rules.

## Philosophy

Read:
- vision.md
- house-eleven-principles.md

before making architectural decisions.

## Workflow

Never implement multiple major features.

Always:

Experience

↓

Architecture

↓

Implementation

↓

Review

↓

Commit

↓

Push

## Design

Always use existing reusable systems.

Never duplicate components.

Prefer extending existing architecture.

## Motion

Motion should be calm.

Never animate for decoration.

## Code

Prefer reusable components.

Avoid one-off implementations.

Never refactor working systems unless there is a clear architectural benefit.
