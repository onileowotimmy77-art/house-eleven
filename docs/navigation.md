# House Eleven Navigation

> Navigation should quietly guide, never interrupt.

---

# Philosophy

Navigation is not a menu.

It is an orientation system.

Visitors should always know where they are without feeling overwhelmed.

The navigation should feel like a concierge rather than a directory.

---

# Core Principles

- Calm over busy
- Clarity over quantity
- Consistency over novelty
- Confidence over promotion

---

# Structure

Primary Navigation

- Home
- Collections
- Journal
- Archive
- About

Secondary Actions

- Search
- Shopping Bag

Avoid adding links that dilute the primary journey.

---

# Behaviour

The navigation should feel stable.

It should never jump unexpectedly.

Transparency, blur and visibility should transition smoothly.

Every state should feel intentional.

---

# Scroll Behaviour

Scrolling should strengthen orientation.

Navigation should adapt gently without becoming the focus.

Avoid abrupt appearance or disappearance.

---

# Mobile Navigation

Mobile deserves the same level of refinement as desktop.

The mobile menu should feel like entering another room within the House.

Not opening an application drawer.

---

# Future Expansion

As the House grows, navigation should evolve carefully.

New destinations should only be introduced when they strengthen the overall experience.

Navigation should remain simple regardless of how much the House expands.

---

# Final Principle

The visitor should remember the journey, not the menu.
